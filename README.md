## 👨‍💻 About Me

I'm a **Mobile Application Engineer** who builds full-stack products end to end — from pixel-perfect Flutter/React Native UIs down to the Postgres schema and payment integration behind them. I care about clean architecture, scalable state management, and shipping things that actually work in production.

- 🔭 Currently building **Drivo** — a full-stack, Uber-style ride-sharing app (React Native + Expo + Stripe + Google Maps)
- 🧠 Deep experience with **Clean Architecture, MVVM & the Repository Pattern** across Flutter and React Native
- 🎨 Built **ECometa**, an AI-powered visual search shopping app — 30k+ LOC, 240+ commits
- 🌱 Always exploring new corners of the mobile & full-stack ecosystem
- 📫 Reach me at **ahmedmahershaaban@gmail.com**
